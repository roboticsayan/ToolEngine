## Local Development :construction:

#### Cloning the repository.
```ssh
git clone https://github.com/roboticsayan/ToolEngine.git
```

#### Installing Dependencies
```ssh
npm install
```

#### Starting Development Server
```ssh
npm start
```

#### Deploy Production
```ssh
npm run build
```
